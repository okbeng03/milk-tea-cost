# 奶茶成本系统
> 根据配方、物料入库，动态精确计算每杯奶茶成本，进而根据每日订单明细计算出每日成本、收益、利润，同时以此为依据指导定价和活动方案。

## 功能模块
1. 菜品
	1. 菜单录入
	2. 配方录入
	3. 成本计算
2. 物料入库
  1. 入库价高提醒
3. 报表
	1. 菜品成本
	2. 订单明细
	3. 每日收益
